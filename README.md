# task-manager-app

Sistema de gerenciamento de tarefas com backend em Java (Spring Boot).

## Estrutura inicial criada

```text
task-manager-api
 ┣ src/main/java/com/taskmanager
 ┃ ┣ controller
 ┃ ┣ service
 ┃ ┃ ┗ impl
 ┃ ┣ repository
 ┃ ┣ dto
 ┃ ┣ entity
 ┃ ┣ config
 ┃ ┗ security
 ┣ src/main/resources
 ┣ src/test
 ┣ pom.xml
 ┣ Dockerfile
 ┗ docker-compose.yml
```

## Stack do backend

- Java 21
- Spring Boot
- Spring Security (JWT)
- JPA / Hibernate
- Maven
- JUnit
- PostgreSQL
- Docker
